# Daily Diary

## Project Overview
This project is a pixel-perfect frontend implementation based strictly on the provided Figma design.  
The goal is to convert the UI/UX design into a functional website without changing layout, colors, or structure.

## Technologies Used
- HTML5
- CSS3 
- JavaScript

## Pages Implemented
- Home Page
- Single Entry Page
- About Page

## Features
- Pixel-perfect UI from Figma
- Responsive layout 
- Clean and structured code
- Dynamic single entry handling using JavaScript
